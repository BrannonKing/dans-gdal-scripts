Dan's GDAL scripts

This package contains the following programs:
  gdal_contrast_stretch   Auto contrast stretch and coversion from 16-bit to 8-bit
  gdal_dem2rgb   Generate hillshaded images from DEMs
  gdal_landsat_pansharp   Pansharpening - works best for Landsat 7 images
  gdal_list_corners   Ouput geometry information in YAML and WKT format - can inspect image to find polygon bounds
  gdal_raw2geotiff   Convert raw binary data to GeoTIFF

Download: dans-gdal-scripts-0.03.tar.gz
